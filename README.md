# MasterMind PWA

## Install and offline test

1. Deploy the contents of this folder to GitHub Pages.
2. Open the site once while online so the service worker can cache the app shell.

### Install on iPhone

1. Open the site in Safari.
2. Tap `Share`.
3. Choose `Add to Home Screen`.

### Install on Android

1. Open the site in Chrome.
2. Open the Chrome menu.
3. Choose `Install App`.

### Verify offline mode

1. Open the app once online.
2. Turn on Airplane Mode.
3. Launch the installed app from the home screen.
4. Confirm the app still opens and gameplay plus the leaderboard still work offline.
